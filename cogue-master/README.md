Cogue
=======

A simple system to automate VASP calculations with queueing systems. See more details in  http://atztogo.github.io/cogue/.

Installation
=============
Build the cogue package from the setup.py file and place the cogue directory inside PyClex like Pyclex/cogue similarly for CPyMonC.

Do ensure that _xtalcomp.so and _spglib.so are aslo placed in PyClex/cogue.
